💾 DATABASE ARCHITECTURE
───────────────────────────────────────────────────────────────────────────

Chroma DB (Vector Database)
├── Collection: credit_documents
│  ├─ Stores document chunks (1000 chars each)
│  ├─ Enables semantic similarity search
│  ├─ Indexed by cosine distance
│  │
│  └─ Fields per chunk:
│     ├─ id: "doc_id_chunk_n"
│     ├─ document: text content
│     ├─ metadata:
│     │  ├─ doc_id
│     │  ├─ company_name
│     │  ├─ document_type
│     │  ├─ file_name
│     │  ├─ chunk_index
│     │  ├─ total_chunks
│     │  └─ timestamp
│     └─ embedding: auto-generated (384 dims)
│
└── Collection: document_metadata
   ├─ Stores complete document information
   ├─ Enables filtering and retrieval
   │
   └─ Fields per document:
      ├─ id: "doc_id"
      ├─ metadata:
      │  ├─ doc_id
      │  ├─ company_name
      │  ├─ document_type
      │  ├─ file_name
      │  ├─ content_length
      │  ├─ chunk_count
      │  ├─ upload_timestamp
      │  ├─ cibil_score (auto-extracted)
      │  ├─ revenue
      │  ├─ bank_inflow
      │  ├─ company_age_years
      │  ├─ litigation_count
      │  ├─ gst_revenue
      │  └─ custom fields
      └─ embedding: auto-generated

Storage: ./chroma_db/ (persists across restarts)

🔧 API REFERENCE (Quick Examples)
───────────────────────────────────────────────────────────────────────────

UPLOAD DOCUMENT
───────────────
from rag_manager import get_document_manager

doc_mgr = get_document_manager()
result = doc_mgr.upload_pdf(
    pdf_file="annual_report.pdf",
    company_name="Acme Corp",
    document_type="Annual Report"
)
print(f"Doc ID: {result['doc_id']}")

SEARCH DOCUMENTS
────────────────
results = doc_mgr.query_documents(
    query="What is company revenue?",
    company_name="Acme Corp",
    top_k=3
)

for r in results:
    print(f"Source: {r['metadata']['file_name']}")
    print(f"Content: {r['content'][:200]}...")

UPDATE DATA
───────────
doc_mgr.update_document_data(
    doc_id="doc_123...",
    updates={
        "verified_revenue": "₹500Cr",
        "risk_level": "LOW"
    }
)
